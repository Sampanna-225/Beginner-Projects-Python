# Python Plotting Project

This project demonstrates how to create a simple line plot using Matplotlib in Python.

## Requirements

- Python 3.x
- Matplotlib

Install dependencies:
```
pip install -r requirements.txt
```

## Usage

Run the script:
```
python Plot.py
```

This will generate a plot and save it as `plot.png`.

## Description

The script plots y = x^2 for x from 1 to 5 and saves the graph as an image file.
